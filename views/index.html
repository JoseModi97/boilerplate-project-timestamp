TimeCraft API

A simple and powerful API for timestamp conversions.

API Endpoints

Get Current Time

GET /api/

Returns the current timestamp in both Unix and UTC formats.

Get Specific Time

GET /api/:date?

Returns the timestamp for a given date string or Unix timestamp.

Example Usage:

  • /api/2024-07-20
  • /api/1721433600000

Example Output

For a valid date, the API returns:

{
  "unix": 1721433600000,
  "utc": "Sat, 20 Jul 2024 00:00:00 GMT"
}

For an invalid date, the API returns:

{
  "error": "Invalid Date"
}
